# docs

The implementation follows the official **Icom IC-9700 CI-V Reference Guide**
(Mar. 2021 revision, doc A7508-3EX). That PDF is Icom's copyrighted material
and is **not** committed to this repo.

To get it, download "IC-9700 CI-V Reference Guide" from Icom's support pages:

- <https://www.icomjapan.com/support/manual/2161/>

Place it here as `IC-9700_CI-V-Reference-Guide.pdf` if you want a local copy
for reference; it is gitignored.

A `screenshot.png` of the panel can be dropped in this folder for the README.
